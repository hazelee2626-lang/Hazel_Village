▸ 처음 설정하는 방법 (1회만)
  1. console.firebase.google.com → 프로젝트 만들기
  2. Authentication → Sign-in method → Google 사용 설정
  3. Firestore Database → 데이터베이스 만들기
  4. 프로젝트 설정 ⚙️ → 웹 앱(</>) 추가 → firebaseConfig 값을 이 파일 상단 FIREBASE_CONFIG에 붙여넣기
  5. Authentication → Settings → 승인된 도메인에 배포 주소(예: xxx.vercel.app) 추가
Firestore 규칙(보안)은 이렇게: rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /villages/{uid} { allow read, write: if request.auth != null && request.auth.uid == uid; } } } ⚠️ 로그인은 파일을 직접 열면(file://) 안 되고, Vercel 배포 주소에서만 동작해.
